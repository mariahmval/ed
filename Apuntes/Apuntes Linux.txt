Terminal:

history: Muestra el historial de comandos.
ctrl-r: Hace una búsqueda en el historial de comandos utilizados.
apt full-upgrade: Actualiza todos los paquetes, incluso aquellos marcados para ser eliminados.
add-apt-repository ppa:[Repositorio]: Añadir repositorio a la lista.
ls [-l]: Lista los archivos en el directorio [en modo de lista].
cat [archivo]: Muestra el contenido de un archivo.
[comando] | grep [caracteres]: Filtra los resultados del primer comando.
passwd [usuario]: Establecer contraseña de usuario.
cd -: Volver al directorio en el que estaba anteriormente, dos puntos (..) lleva al directorio superior.
ctrl+z: Cancela la ejecución del comando actual.
q: Cierra la lista mostrada

Java:

Comprobar versión instalada: java -version
Añadir repositorio Java Oracle: sudo add-apt-repository ppa:webupd8team/java
Instalar jdk8 de java: apt install oracle-java8-installer
Compilador java: javac
Ejecutar Clases java: java [Clase (no el nombre del archivo)]

Nautilus:

Para ver las carpetas en árbol:
Editar > Preferencias > Visualización > Navegar por las carpetas en un árbol

La extensión .tar indica que se trata de un contenedor de archivos y enlaces, si además tiene un .gz quiere decir que está comprimido.

En /home/usuario/.local/share/applications se encuentra la configuración de los accesos rápidos de la barra de tareas del escritorio.
